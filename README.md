# Documentación de Proyecto

## Instalación

A continuación, se detallan los pasos para instalar el proyecto:

1. Clona el repositorio desde GitHub:
    ```bash
    git clone https://github.com/PatricioVargasR/prueba.git
    ```

2. Ingresa al directorio del proyecto:
   - Recomendable que se llame prueba y la ruta quede ~/xampp/htdocs/prueba/...
    ```bash
    cd htdocs
    ```

3. Uso de BD:
   - Importa el archivo **productos.sql** a una base de datos del mismo nombre

## Elementos

En este proyecto, encontrarás los siguientes elementos principales:

- **Módulo USER:** Interfaz donde se podrá interactuar con la tienda y la información de la empresa con la opción de iniciar sesión, registrarse y carrito
- **Módulo ADMIN:** Interfaz de administrador donde se podrá editar las categorías y los productos disponibles.
